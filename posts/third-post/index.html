AI technical analysis teacher

Financial education with AI

For my ongoing master’s thesis, I’m building an AI-powered educational tool that bridges the gap between raw market data and human insight. The goal is to help users understand technical analysis by letting them interact with real-time stock and crypto charts in an intuitive, visual way.

Users are first presented with a live-rendered price chart. By clicking a simple “find patterns” button, the app scans the dataset for key technical indicators - including flags, head & shoulders, and more - then overlays them directly onto the chart. Alongside the visual display, the AI generates a tailored explanation of each pattern, breaking down what it is, why it matters, and how it’s typically interpreted by traders.

The server is built in Python, combining pattern recognition algorithms with the OpenAI API for generating explanations. The client side is a responsive Next.js + React TypeScript interface. The tool currently supports four core patterns and is compatible with virtually any stock or cryptocurrency data.

My role encompasses the full-stack pipeline - from chart parsing and signal extraction to LLM integration and front-end interactivity. I’m especially focused on making complex concepts feel approachable for those new to trading, while maintaining enough depth for intermediate learners.

This project is where finance meets explainability, providing interactive explanations for tiny fraction of the price (1% of a cent per pattern) that an expert advice would cost. It’s still a work in progress - but already showing promise as a learning companion for anyone curious about how patterns shape the markets.
